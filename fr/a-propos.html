À propos du site web

une phote de moi

Bonjour, je m’appelle Patrick Pineault et j’ai 21 ans. En plus, je suis le créateur de ce site web si vous voulez savoir à quoi je ressemble regardez au-dessus de ce paragraphe c’est une photo de moi. Ce site est un circuit touristique de différents points d’intérêts sur un lieu que j’ai décidé d’inventer du nom de Ville des capybaras poilus. Les raisons pour lesquelles j’ai décidé de choisir ce fameux c’est parce que je me suis inspiré de l’algorithme quand je consulte mon Instagram. J’y consulte beaucoup de photos et de vidéos sur les capybaras Dans les phrases suivantes, je vais vous expliquer mes inspirations pour les trois lieux qui sont présentés dans la page point d’intérêt. Pour Capyzoo : C’est le Parc Omega à Montebello, zoo de Granby à Granby, et le Parc Safari à Hemmingford. Pour Capyaqua : C’est le Bora Parc à Valcartier et le Super-Aqua Club à Pointe-Calumet. Pour Capyart : C’est le musée du Domaine à Pohénégamook, (il n’existe plus aujourd’hui). Pour les photos dans le carrousel, j’ai demandé à mon père d’être le concepteur et le photographe pour les photos, puisqu’il est passionné de photographie et il connaît bien le logiciel PaintShop Pro.
